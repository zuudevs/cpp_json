# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| 1.0.x   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within `cpp_json`, please do **not** create a public GitHub issue.

Instead, please send an email to **zuudevs.gmail.com**. We will review it and respond within 48 hours.
